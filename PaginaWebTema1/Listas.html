Etiquetas de Listas

NOTA: Las listas HTML permiten a los desarrolladores web agrupar un conjunto de elementos relacionados en listas.

1Listas HTML desordenadas.

Una lista desordenada comienza con con la etiqueta.<ul><li> Los elementos de la lista se marcarán con viñetas (pequeños círculos negros) de forma predeterminada:

2Listas HTML ordenadas

Una lista ordenada comienza con la etiqueta <ol><li>. Los elementos de la lista se marcaran de forma predeterminada con numeros:

3Listas de descripcion HTML

HTML también admite listas de descripción. Una lista de descripción es una lista de términos, con una descripción de cada término La etiqueta define la lista de descripción, la etiqueta define el término (nombre) y la etiqueta describe cada término:<dl><dt><dd>:
