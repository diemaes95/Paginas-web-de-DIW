¿Que son las tecnicas ?

Las Técnicas para las Pautas de Accesibilidad al Contenido en la Web 1.0 (traducción al castellano de Techniques for Web Content Accessibility Guidelines 1.0) describen de forma concreta y con ejemplos prácticos cómo diseñar las páginas web para que se cumplan los requisitos de accesibilidad.

Nos podemos encontrar divididas estas tecnicas en ramas como: tecnicas generales, aria, fallos comunes, tecnicas css, tecnicas pdf y mas tecnicas que podemos encontrar...
