Etiquetas de Formulario

1Elemento <form>

El elemento <form> es un contenedor para diferentes tipos de elementos de entrada, tales como: campos de texto, casillas de verificación, botones de opción, botones de envío, etc.

2Elemento <Imput>

Es el elemento de formulario mas utilizado. y se puede mostrar de muchas maneras, dependiendo del atributo por ejemplo:

Campo de texto:

El <input type="text">define un campo de entrada de una sola línea para la entrada de texto.

Botones de radio:

El <input type="radio">define un boton de radio . Los botones de radio permiten al usuario seleccionar UNA de un número limitado de opciones.

Casillas de verificacion:

Las casillas de verificación <input type="checkbox"> permiten a un usuario seleccionar CERO o MÁS opciones de un número limitado de opciones.

El boton enviar:

Define un botón para enviar los datos del formulario a un controlador de formularios.<input type="submit"> Se suele usar para enviar datos a el servidor donde los ejecuta un archivo php . El documento php se especifica en el atributo action de el formulario.

El atributo Method de el formulario:

El atributo Method especifica el método que se utilizará al enviar los datos del formulario. Los datos del formulario se pueden enviar como variables declarandolas en el atributo url y el tipo de metodo en method <form url="datos" method="get"method="post"> El método predeterminado al enviar datos de formulario es GET.

El atributo action:

El atributo action lo que hace es referenciar el archivo php donde se van a ejecutar el código
